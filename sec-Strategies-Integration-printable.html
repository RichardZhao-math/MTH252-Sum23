    Worksheet 1.1 Strategies to Evaluate Integrals
    

    Back in the 1990s, an instructor placed the following integral at the beginning of a final exam in an integral calculus course:
    🔗
    \begin{equation*} \int \frac{x}{x^2 - 9}\, dx \end{equation*}

    1.

    Imagine you were a student taking this exam. Based on your current knowledge, what is the first method you would try to evaluate this integral? Clearly state your reasoning for choosing this method as your first approach. Then, carry out the integration using this method and show all steps clearly.
    🔗
    🔗

    2.

    Now, come up with a different method that could also be used to evaluate the same integral. Briefly explain why you believe this alternative method is valid and likely to succeed. Then, evaluate the integral using this second method, again showing all steps clearly.
    🔗
    🔗
    When you finished the above question, expand the solution below and check your answer.
    🔗
    Solution.
    There are actually three methods to solve this integral. They are \(u\)-sub, trig sub, and particial fraction decomposition.
    🔗
    1. Using \(\boldsymbol{u}\)-sub: Let \(u = x^2 - 9\text{.}\) Then \(du = 2x\, dx\text{.}\)
      🔗
      \begin{align*} \int \frac{x}{x^2 - 9}\, dx \amp = \int \frac{1}{2u}\, du \\ \amp = \frac{1}{2}\ln|u| + C \\ \amp = \frac{1}{2}\ln\left|x^2 - 9\right| + C \end{align*}
      🔗
    2. Using Trig sub: Let \(x = 3\sin(\theta)\text{.}\) Then \(dx = 3\cos(\theta)\, d\theta\text{.}\)
      🔗
      \begin{align*} \int \frac{x}{x^2 - 9}\, dx \amp = -\int \frac{x}{9 - x^2}\, dx \\ \amp = -\int \frac{3\sin(\theta)}{9 - 9\sin^2(\theta)}\cdot 3\cos(\theta)\, d\theta \\ \amp = -\int \frac{9\sin(\theta)\cos(\theta)}{9\cos^2(\theta)}\, d\theta \\ \amp = -\int \tan(\theta)\, d\theta \\ \amp = -\ln|\sec(\theta)| + C_0 \\ \amp = -\ln\left|\frac{3}{\sqrt{9 - x^2}}\right| + C_0 \\ \amp = -\left(\ln(3) - \ln\left|9 - x^2\right|^{\frac{1}{2}}\right) + C_0\\ \amp = \frac{1}{2}\ln\left|9 - x^2\right| - \ln(3) + C_0\\ \amp = \frac{1}{2}\ln\left|x^2 - 9\right| + C \end{align*}
      where \(C = -\ln(3) + C_0\text{.}\)
      🔗
      🔗
    3. Using Partial Fraction: Let \(A\) and \(B\) be two numbers. Note that
      🔗
      \begin{equation*} \frac{x}{x^2 - 9} = \frac{x}{(x + 3)(x - 3)} = \frac{A}{x + 3} + \frac{B}{x - 3} \end{equation*}
      This implies that
      🔗
      \begin{equation*} x = A(x - 3) + B(x + 3) \end{equation*}
      We can determine that \(A = \dfrac{1}{2}\) and \(B = \dfrac{1}{2}\text{.}\)
      🔗
      \begin{align*} \int \frac{x}{x^2 - 9}\, dx \amp = \int \left(\frac{1/2}{x + 3} + \frac{1/2}{x - 3}\right)\, dx \\ \amp = \frac{1}{2} \int \left(\frac{1}{x + 3} + \frac{1}{x - 3}\right) \, dx \\ \amp = \frac{1}{2} \left(\ln|x + 3| + \ln|x - 3|\right) + C \\ \amp = \frac{1}{2}\ln\left|x^2 - 9\right| + C \end{align*}
      🔗
    🔗
