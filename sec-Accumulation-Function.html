    Section 1.1 The Accumulation Function

    Let us first define what the accumulation function is! The definition is taken from your textbook page 326 (but your textbook call it the area function).
    🔗

    Definition 1.1.1.

    Let \(f\) be a continuous function on an open interval \(I\) and let \(a\) be in \(I\text{.}\) We define the accumulation function as
    \begin{equation} A(x) = \int_a^x f(t)\, dt\tag{1.1.1} \end{equation}
    🔗
    🔗
    Using our knowledge of function notations, we know that the name of the accumulation function is \(A\text{,}\) with the input of \(x\) and the output of \(A(x)\text{.}\)
    🔗

    Exercises Part (a): An example

    Let \(f(t) = 2t - 1\) and \(a = 1\text{.}\) Then we can define the accumulation function here to be
    \begin{equation*} A(x) = \int_1^x \left(2t - 1\right)\, dt \end{equation*}
    Compute \(A(1)\text{,}\) \(A(2)\text{,}\) \(A(3)\text{,}\) and \(A(4)\text{.}\) Then graph \(y = f(t)\) and indicate what these values represent on the graph.
    🔗
    Hint.
    Julie went through a similar example in one of the lecture videos in this week. If you get stuck or you want to confirm that you are on the right track, find that example.
    🔗
    🔗
    Solution.
    Below is the graph of \(f(t) = 2t - 1\text{.}\)
    🔗
    \(A(1)\) represents the area under the line \(y = f(t)\) from \(t = 1\) to \(t = 1\text{.}\) Graphically speaking, the area is 0, shown in the diagram below.
    🔗
    \(A(2)\) represents the area under the line \(y = f(t)\) from \(t = 1\) to \(t = 2\text{.}\) Graphically speaking, the enclosed region is a trapezoid, with the area of
    \begin{equation*} A(2) = \frac{1(f(1) + f(2))}{2} = \frac{1 + 3}{2} = 2 \end{equation*}
    The diagram is shown below.
    🔗
    \(A(3)\) represents the area under the line \(y = f(t)\) from \(t = 1\) to \(t = 3\text{.}\) Graphically speaking, the enclosed region is a trapezoid, with the area of
    \begin{equation*} A(3) = \frac{2(f(1) + f(3))}{2} = \frac{2(1 + 5)}{2} = 6 \end{equation*}
    The diagram is shown below.
    🔗
    \(A(4)\) represents the area under the line \(y = f(t)\) from \(t = 1\) to \(t = 4\text{.}\) Graphically speaking, the enclosed region is a trapezoid, with the area of
    \begin{equation*} A(4) = \frac{3(f(1) + f(4))}{2} = \frac{3(1 + 7)}{2} = 12 \end{equation*}
    The diagram is shown below.
    🔗

    Exercises Part (b): Meaning of the function

    Functions have inputs and outputs. What does the input of the accumulation function, \(x\text{,}\) represent and what does the output of the accumulation function, \(A(x)\text{,}\) represent? Be as specific as possible in your answer.
    🔗
    After you identified the meaning of the inputs and outputs, try to come up with a reason why we call this the accumulation function?
    🔗
    Solution.
    Notice that the variable \(x\) appears to be in the place of the upper limit of integration in the integral. This implies that the input of the accumulation function, \(x\text{,}\) represents the upper limit of integration. It tells us how "far" we are integrating from the starting point of \(t = a\text{.}\)
    🔗
    The output of the accumulation function, \(A(x)\text{,}\) represents the signed area under the curve of \(y = f(t)\) from \(t = a\) to \(t = x\text{.}\)
    🔗
    We call \(A\) the accumulation function because it accumulates the signed area under the curve as we move from the starting point of \(t = a\) to some later point \(t = x\text{.}\) To make it more generalized, the accumulation function accumulates some quantity (and it doesn’t have to be some area) between two points. We will look at a more generalized version of integrals in week 5 investigation.
    🔗

    Exercises Part (c): Structure of the formula

    Notice that there are two variables in the accumulation function, \(x\) and \(t\text{.}\) Explain why we need both \(x\) and \(t\) to define the accumulation function?
    🔗
    For example, why wouldn’t it make sense to define the accumulation function using ONLY \(x\) as the variable?
    🔗
    Hint.
    Let’s say we define the accumulation using only \(x\) as the variable like this:
    \begin{equation*} A(x) = \int_a^x f(x)\, dx \end{equation*}
    Then how would we compute, let’s say \(A(2)\text{?}\) Well, we will need to replace ALL the \(x\)’s with \(2\text{...}\) Does the expression make sense then?
    🔗
    🔗
    Solution.
    The short answer is that \(x\) and \(t\) do two different jobs since there are two types of functions here:
    • \(x\) is the input variable of the accumulation function, which represents the upper limit of integration of the integral.
      🔗
      🔗
    • \(t\) is just a dummy variable we use to represent the input of the function \(f\text{.}\)
      🔗
      🔗
    🔗
    If we define the accumulation function only using \(x\) as an variable like
    \begin{equation*} A(x) = \int_a^x f(x)\, dx \end{equation*}
    Then the expression wouldn’t make sense since it is impossible to evaluate this function.
    🔗
    As an example, \(A(2)\) would be
    \begin{equation*} A(2) = \int_a^2 f(2)\, d2 \end{equation*}
    Well... \(f(2)\) is a constant (the output of the function with an input of 2) so we are really integrating a number here (boring). Also, what is \(d2\text{...}\) Well we know that the symbol \(d\) here represents the infinitesimally small amount of change. But 2 is a constant and the value of 2 will never change... So \(d2\) doesn’t really make sense mathematically.
    🔗
